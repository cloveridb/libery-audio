Hubungkan Roblox
Cara mendapatkan API Key:
1. Buka Roblox Creator Hub → API Keys
2. Klik Create API Key → beri nama bebas
3. Permissions: Assets:Read dan Assets:Write
4. Save → salin key yang muncul (hanya tampil sekali!)
Kosongkan jika tidak ingin mengubah key yang tersimpan
🔒 API Key dienkripsi AES-256
